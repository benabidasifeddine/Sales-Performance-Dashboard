# 📊 Sales Performance Report | Power BI

## 📌 Project Overview

This project is an interactive **Sales Performance Report** built with **Microsoft Power BI**.  
The objective is to analyze sales performance, profitability, customer behavior, and business trends to support data-driven decision-making.

---

## 🎯 Business Objectives

The dashboard answers key business questions:

- How are sales and profits evolving over time?
- Which products and categories generate the highest revenue?
- Which regions perform best?
- Are sales trends increasing, stable, or decreasing?
- Which segments require attention to improve profitability?

---

## 🛠️ Tools & Technologies

- **Power BI Desktop**
- **DAX** (Data Analysis Expressions)
- **Power Query** (Data Transformation)
- **Excel / CSV Dataset**

---

## 📈 Key Metrics (KPIs)

The dashboard includes:

- Total Sales
- Total Profit
- Profit Margin %
- Total Orders
- Average Order Value (AOV)
- Total Quantity Sold
- Sales YTD
- Sales MTD
- Previous Year Sales
- YoY Growth %

---

## 📊 Dashboard Features

### Sales Overview
- Sales performance tracking
- Revenue trends over time
- Comparison with previous periods

### Product Analysis
- Best-performing products
- Category profitability
- Contribution to total sales

### Regional Analysis
- Sales distribution by region
- Identification of high and low-performing areas

### Profitability Analysis
- Profit margin evaluation
- Identification of profitable and underperforming segments

---

## 🧮 Data Modeling

The project uses a **star schema approach**:

**Fact Table**

**Dimension Tables**
- Date
- Product
